# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/black-pearl.git
```

Go to the project directory

```bash
  cd black-pearl

```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
